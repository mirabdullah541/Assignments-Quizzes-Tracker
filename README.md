# 📚 Class Assignments & Quizzes Tracker

A real-time, shared web app for university students to collaboratively track assignments and quizzes — no login required.

---

## 🌐 Live App

👉 **[Open the App](https://mirabdullah541.github.io/Assignments-Quizzes-Tracker/)**

---

## 📌 Overview

**Class Assignments & Quizzes Tracker** is a single-page web application where every student in a class can post, view, and track upcoming assignments and quizzes. All data syncs live across every device — the moment someone adds an item, the whole class sees it.

---

## ✨ Features

- 📋 **Assignments & Quizzes** — support for both types, clearly labeled
- 🟢 **Auto Status** — items are automatically marked Ongoing, Ending Soon, or Ended based on the deadline
- ☁️ **Real-time Sync** — powered by Firebase Firestore, changes appear instantly for everyone
- 🔍 **Search** — find items by title or subject
- 🗂️ **Filter & Sort** — filter by subject, type, or status; sort by deadline or subject
- 🌙 **Dark / Light Mode** — toggle between themes
- 📱 **Fully Responsive** — works on phones, tablets, and desktops

## 🤝 How to Use

1. Open the app link in any browser
2. Click **"+ Add"** to post a new assignment or quiz
3. Fill in the subject, title, source, deadline, and optional notes
4. Hit **"Add for Class"** — it appears for everyone instantly
5. Click any card to view full details or delete it
